# Server-Multi-Room-Chat
This is a simple application of TCP/IP protocol where a user have a server and he could create multiple clients on the same port and they can use it to communicate between themselves.

How to use it ?

Step 1 : Simply Download this project.

Step 2 : Initialize the server by simply running python server.py in your terminal

Step 3 : Enter you Server address and port address into your browser and it would start working
